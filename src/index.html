Om uppgiften

Uppgiften går ut på att skapa en demowebbplats med hjälp av automatiserande verktyg. GitHub ska användas kontinuerligt och versionshantering likaså. GitHub ska även användas för att automatisera publiceringsprocessen.


.gitignore-fil ska skapas för att exkludera filer som inte ska versionhanteras.


Webbplatsen ska ha åtminstone tre undersidor:

  • Startsida med info om uppgiften(denna sida)
  • Undersida för bilder med minst fem stycken fotografier
  • En undersida för publicering där du beskriver din skapande arbetsprocess.

Punktlista för vad uppgiften ska innehålla:

  • Minifiering av CSS
  • Transpilering och minifiering av JavaScript
  • Minifiering av HTML
  • Optimering av bilder automatiskt vid uppdatering av bilder till Webp, avif eller liknande.
  • Bakåtkompabilitet för att stödja så många webbläsare som möjligt.
  • Kopiering av arbetsfiler till mapp för publicering med optimerade filer.
